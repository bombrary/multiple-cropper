Multiple Cropper

Description

  • Add button: add box.
  • Delete key: delete selected box.
  • Click: select box.
  • Drag: move or scale box.
  • Arrow Key: move selected box.
  • Copy button: copy selected box.
  • Load image button: load image, allowed png, jpeg, gif, and bmp.
  • Right side: cropped images for each boxies. Right click to save a image.
  • Download button: download all cropped images as zip file.
